# Music AI Vault

## Overview
This repository is an Obsidian vault containing open-source GDPR compliance guidelines for AI in music. It focuses on data privacy, fairness, accessibility, and aesthetics, curated for musicians, engineers, and creatives. The main document outlines requirements and constraints for ethical AI-music applications.

Key features:
- Human-centered design principles integrated with regulatory compliance.
- Aesthetic and accessible formatting for diverse users.
- AI-assisted curation with substantial human refinements.

## How to Use
1. **Install Obsidian**: Download and install Obsidian from [obsidian.md](https://obsidian.md/).
2. **Clone the Repo**: Run `git clone https://github.com/Pi-Kay/Music-AI-Vault.git` in your terminal, or download as ZIP.
3. **Open in Obsidian**: Launch Obsidian, select "Open folder as vault", and choose the cloned folder.
4. **View Main Content**: Open `Music AI Document.md` for the full guidelines. Use Obsidian's graph view for interconnected notes.

If you're not using Obsidian, view the raw Markdown files directly on GitHub (e.g., [Music AI Document.md](Music%20AI%20Document.md)).
